## Open Source Standards

- CI validation for build and tests
- PR title policy and PR template
- Issue templates and severity labeling
- Dependency review and CodeQL security scanning
- Automated release pull requests

## Quick Start

1. Install dependencies:

```bash
npm ci
```

2. Start development server:

```bash
npm run dev
```

3. Run quality checks:

```bash
npm run build
npm test --if-present
```
