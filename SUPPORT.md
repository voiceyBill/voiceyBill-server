# Support

## Getting Help

- For bug reports: use the Bug report issue template.
- For feature ideas: use the Feature request issue template.
- For usage questions: use the Question issue template.

## Before Opening an Issue

- Search existing issues first.
- Make sure you are on a recent version of the default branch.
- Include reproduction steps, logs, and environment details.

## Response Expectations

This is an open source project maintained in available time. We aim to respond
to new issues as quickly as possible, but response times can vary.
